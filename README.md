# 10-Digit-UP-Counter-
This project demonstrate 10-digit UP counter  implemented by ESP32 microcontroller, displaying numbers from 0 to 9999999999 on CLCD (Character Liquid Crystal Display) ,The Counter increments every iteration, extracting each digit and displaying it on the display. 
Features:
CLCD interface
simple and easy-to-use code.
Hardware Requirements:
Microcontroller(ESP 32 DEV WROOM KIT)
CLCD (16X2)
Software Requirements:
Arduino IDE;
WOKWI
